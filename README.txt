Este proyecto funciona por medio de una api rest, creada con angular.

Proyecto realizado con el Framework Ionic, 
para poder compilar y ejecutar este mismo es necesario hacer la previa
instalacion de ionic en tu computadora y correr el servidor en angular.
De igual manera tener instalado MongoDB para tu bases de datos.

El proyecto se corre con el comando:

ionic serve : Para abrirlo desde un explorador

Si quieres instalarlo en un dispositivo o emulador
correr el siguiente comando

ionic cordova run android
